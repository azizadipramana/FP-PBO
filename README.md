### Deskripsi Project
Project ini merupakan aplikasi kuis interaktif berbasis Java yang dibuat menggunakan Java Swing sebagai GUI dan menerapkan konsep-konsep Pemrograman Berorientasi Objek (PBO).
Aplikasi ini menampilkan antarmuka pengguna berupa halaman login, halaman aturan, halaman kuis, hingga halaman hasil akhir yang menunjukkan skor dan koreksi jawaban.

Pengguna memasukkan nama, membaca rules, lalu mengerjakan 10 soal pilihan ganda dengan timer. Sistem akan menghitung skor secara otomatis dan memberikan laporan lengkap jawaban benar maupun salah.

### Fitur Utama Aplikasi
#### 1. Login Screen
- Pengguna memasukkan nama.
- Tersedia tombol Start dan Rules.
- Menampilkan gambar/logo di bagian kiri.
#### 2. Rules Screen
- Menampilkan aturan pengerjaan kuis.
- Tombol Start untuk mulai mengerjakan kuis.
- Tombol Back kembali ke halaman login.
#### 3. Quiz Screen
- Menampilkan 10 soal pilihan ganda.
- Setiap soal terdiri dari 4 opsi.
- Timer 15 detik per soal.
- Tombol navigasi:
  - Next → lanjut ke soal berikutnya
  - 50-50 Lifeline → menghilangkan dua opsi yang salah
  - Submit → mengakhiri kuis dan menampilkan hasil
#### 4. Score Screen
- Menampilkan skor akhir berdasarkan jawaban.
- Menampilkan gambar “Good” atau “Bad” tergantung skor.
- Menampilkan laporan lengkap:
  - Soal ke berapa yang benar
  - Jawaban user
  - Kunci jawaban
- Tombol Main Lagi untuk kembali ke login.
